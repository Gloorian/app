Googlê Password Generator

La solution à tous vos problèmes de mémoire.

Ce site permet de générer très facilement des mots de passe complexes, uniques, adaptés à tous vos besoins, et qu’il vous sera difficile d’oublier !

Entrez simplement l’adresse du site web sur lequel vous désirez vous inscrire, définissez un « Mot de passe principal » puis lancez vous !


Googlê Password Generator est une production Arnaud Lefort basée sur « Stanford PwdHash ».
