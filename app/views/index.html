Bienvenue à {{title}}, un site/vitrine des applications que l'on peut réaliser avec Node.js. À l'heure actuelle, il en regroupe quatre: un réducteur d'url, un hébergeur de fichiers, un générateur de mots de passe sécurisés et un hébergeur de code.

Node.js

Ce site est entièrement fait à l'aide de Node.js, une API qui permet l'utilisation de javascript côté serveur. Node.js est un "equivalent" de Apache et PHP qui est orientée requêtes asynchrones. Bien que Node.js soit en général plutôt préféré pour les applications en temps réel, son utilisation pour des applications classiques est tout aussi simple.

Modules

Les modules utilisés pour réaliser ce site sont en majorité Matador, un framework pour Node.js auquel j'ai attribué mon organisation MVC, Mysql, un module afin de gérer la base de donnée et Hogan.js, un moteur de template inspiré de Mustache.

Librairies et Framework

Les librairies utilisées côté client sont Mootools, une librairie permettant la gestion simple du DOM, des requêtes AJAX (voir le réducteur d'url) et des transitions CSS, et enfin Bootstrap de Twitter, un framework HTML/CSS qui m'a permis de faire le design de ce site web très simplement.
